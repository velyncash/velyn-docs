# Cards Specification

## Card Types

### Virtual Cards
- Instantly issued
- Online usage
- Controlled via app

### Physical Cards (Planned)
- Offline & in-store payments
- Same security model as virtual cards

---

## Card Controls

Users can:
- Enable / disable cards
- View card history
- Lock card instantly

---

## Security Model

- Card usage validated by backend
- PIN confirmation for sensitive actions
- Immediate disable in case of misuse
