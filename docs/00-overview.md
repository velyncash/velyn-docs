# Overview

**Velyn.Cash** is a modern neobank & payments experience built for daily use—focused on **clarity**, **security-by-design**, and a clean **white/pink glass UI**.

This repository documents:
- What Velyn.Cash is and what it’s building
- Core features and user flows
- High-level system architecture
- Security principles
- Roadmap and FAQs

## What Velyn.Cash Delivers

Velyn.Cash is designed around practical financial actions:
- Sign in with a simple, familiar access model (username/password + PIN)
- Pay in real-world contexts (QR, contactless/NFC, card-based flows)
- Track spending and budgets without finance jargon
- Maintain trust through transparent development and documented progress

## Design Principles

- **Clarity first:** Minimal UI, readable language, predictable flows  
- **Security by design:** Sensitive actions require intentional confirmation  
- **Everyday utility:** Payments, cards, budgeting, analytics—no unnecessary noise  
- **Build in public:** Visible progress through repositories and documentation  

## Development Status

Velyn.Cash is in **active development**.  
Documentation and specifications evolve alongside implementation.
