# Frequently Asked Questions (FAQ)

This section answers common questions about **Velyn.Cash**.
The goal is to provide **clear, honest, and simple explanations**.

---

## What is Velyn.Cash?

Velyn.Cash is a **modern neobank and payment platform** designed for everyday use.
It focuses on clarity, security-by-design, and real-world payment functionality.

---

## Is Velyn.Cash live?

Velyn.Cash is currently **under active development**.

Some features are available at UI and testing stages, while others are being implemented incrementally.
This is a work in progress, not a finished product.

---

## Why is development public on GitHub?

Velyn.Cash is built **in public** to ensure transparency.

The public GitHub repositories allow users to:
- See real development progress
- Verify that the product is being actively built
- Understand how the system evolves over time

---

## What platforms will Velyn support?

Planned platforms include:
- Web application
- Mobile application (Android & iOS)

Mobile support is a core focus of the platform.

---

## What payment methods will be supported?

Velyn.Cash is designed to support:
- QR payments
- NFC (contactless) payments
- Card-based transactions (virtual and physical)
- RFID-based use cases (controlled environments)

Availability may vary by phase.

---

## Do I need a token to use Velyn.Cash?

Token usage is planned as part of the ecosystem design.
Details about token utility and requirements will be published separately.

Basic app usage is designed to remain accessible.

---

## Is Velyn.Cash secure?

Security is a core design principle.

Key protections include:
- Secure authentication flows
- PIN-based confirmation
- Backend-enforced rules
- Minimal data exposure in client applications

Security features will continue to evolve.

---

## How can I join testing or beta access?

Private beta access is **invitation-based** during early phases.
Public beta access will be announced in later stages.

Updates will be shared through official channels.

---

## Is my financial data shared with third parties?

No.

User data is handled privately and is not shared externally without clear purpose or consent.
Privacy is respected by default.

---

## Will features change over time?

Yes.

Velyn.Cash is actively evolving.
Features may be improved, adjusted, or expanded based on testing and feedback.

---

## Where can I follow updates?

- GitHub: https://github.com/velyncash

Official updates will be shared as development progresses.
